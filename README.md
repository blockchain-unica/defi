# DeFi archetypes in OCaml

This project implements Decentralized Finance (DeFi) archetypes in OCaml.

Usage
-----

To build a test:
```sh
$ dune build test/test.exe
```

To run a test:
```sh
$ dune exec test/test.exe
```
